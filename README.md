> A push-me-pull-ya client for consul + vault


## Install

```sh
$ npm install --save pmpy
```


## Usage

```js
var pmpy = require('pmpy');

pmpy('Rainbow');
```

```sh
$ npm install --global pmpy
$ pmpy --help
```
